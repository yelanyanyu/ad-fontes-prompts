> **Ad Fontes** （拉丁语：回到源头）
> 
> 一个从 **现象学** 与 **具身认知** 视角出发的语言学习项目。我们利用大语言模型，带你超越词表记忆，回归语言被创造和感知的源头。

## 项目缘起

你是否也厌倦了在“`Apple - 苹果`”这样的对应关系中机械记忆？传统“词典记忆法”能帮你通过考试，却难以让你真正**理解**一门语言——像母语者那样不假思索地感受词句的力度与温度。

**Ad Fontes Prompts** 正是为此而生。它不是一套翻译工具，而是一组精心设计的 **“体验重构”提示词**。我们相信，语言是活生生的体验，而非冰冷的信息。因此，我们通过提示词引导AI完成三件事：

1.  **解构**：溯源词根，还原词语背后最本初的**物理动作**与**心理动势**。
2.  **重构**：用第一人称构建**具身叙事场景**，让你“亲身”经历词语的诞生。
3.  **连接**：清晰阐释从字面动作到抽象含义的**逻辑桥梁**，让理解自然发生。

## 核心理念

> 💡 想了解更多背后的思考？请阅读详细的 [理念阐述](docs/PHILOSOPHY.md)。

### 当记“Hammer”时，我们在记什么？

若只记 `Hammer = 锤子`，你只记住了一个静态物品。但若理解 **Hammer**，你感受到的应是：*为建造安身之所而寻找硬物、高举、下砸的**目的**与**力量***。

前者是知识，后者是体验。本项目致力于帮你获得后者。

### 通往“母语者直觉”的路径

母语者的理解源于**场景唤起**，而非翻译转换。

*   ❌ **传统路径**：看到英文 → 在脑中查询中文意思 → 勉强理解
*   ✅ **本项目的路径**：看到英文 → 感知其词源动作 → 在脑海中**重建具身体验** → 获得直觉理解

我们不否定中文或词根词缀的作用。恰恰相反，我们视词根词缀为 **“时空坐标”** ，并用中文卓越的叙事能力，为你模拟出那个专属母语者的、充满意蕴的内心世界。

## 项目能力

目前，提示词主要支持以下两类深度解析：

### 1. 英语词组与习语分析
解析词组中每个词（尤其是介词）的**动态意象**，并串联成完整的心理图景。

**解析逻辑示例：**
*   **核心动词还原**：分析 `ran into` 时，会回溯到 `run` 的原型，聚焦其“快速移动”的本质。
*   **介词动态化**：将 `into` 不视为静态的“里面”，而是看作一个带有力量感和方向性的**动作矢量**（碰撞、进入）。
*   **构建体验场景**：用第一人称叙事，让你感受从“奔跑”到“意外撞入”整个过程的肌肉与心理变化。

### 2. 英语单词深度解析
通过词源，将单词还原到最初被创造的具体情境中。

**经典案例：discuss**
其词根源于“敲打 (`quat-`)”与“分离 (`dis-`)”。我们会重构这样一个场景：*通过反复敲打、摇晃来筛去谷糠，使果实显露。* 于是，`discuss` 就不再是抽象的“讨论”，而是“通过思想碰撞使真理明晰”的生动过程。

## 效果预览

**输入一个词组**：`Run into`

> **🤖 AI 的解析会包含：**
>
> **【逻辑拆解】**
> *   **核心动作**：`Run`（双腿快速交替，充满动能与方向性）。
> *   **动态矢量**：`Into`（不仅是空间上的“进入”，更是带有力道的“撞击”与“闯入”）。
> *   **合成效应**：高速运动 + 突然的碰撞 = 一次预料之外的物理接触。
>
> **【具身体验】**
> *   想象你正专注奔跑，风掠过耳边，视线略显模糊。
> *   陡然间，一个身影闯入你的路径，你来不及减速，整个人“撞入”了对方的空间。
> *   这种身体的急停与冲击，完美对应了生活中与某人**不期而遇**时的那份猝不及防与惊愕。
>
> **【隐喻演变】**
> 从**身体的意外碰撞**，自然延伸到**人际的偶然相遇**。
